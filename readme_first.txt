Задание 1 (node index.js): в принципе в сonsole.log('Record 3'); первая буква с - русская, 
потому и ошибка выскакивала.

Задание 2: коммандная строка должна быть типа "node timer.js 2021-10-30T17:21:00",
те аргумент вводится в формате  ISO8601, в противном случае вылетает ошибка.
Если ввести прошедшее время, то вылезает ошибка "Установлено неверное время таймера!!!"
По истечении времени таймера вылезает сообщение "Время вышло!!!" и программа заканчивает работу.

В принципе без проблем можно добавить любое разумное количество таймеров через "TimersType".
